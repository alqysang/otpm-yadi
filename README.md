OTPM Yadi Tayawi merupakan ekstensi browser untuk mengelola kode Authentication Two-Factor (2FA) dengan mudah dan nyamanan. Ekstensi ini dapat digunakan untuk menghasilkan berbagai kode OTP. Ada beberapa Kode OTP yang akan terisi otomatis ke form input diantaranya dengan nama issuer :
* Dapodik
* SDM
* SIASN
* InfoGTK

### Catatan
Agar Kode OTP dapat terisi otomatis ke form input, pastika nama issuer sama dengan nama issuer di atas. Jika nama tidak sesuai maka Gunakan tombol edit untuk mengubahnya. Pastikan huruf besar dan kecilnya sama, selain dari issuer yang ada, dapat menggunakan [ctrl] + [v]
